## Install the project

Clone this repository:

```
git clone https://github.com/Orphee1/allocine-react.git
cd allocine-react
```

Install packages:

```
npm install
```

When installation is complete, run it:

```
npm start
```

## Technologies

- React - version 16.12
- Axios - version 0.19
